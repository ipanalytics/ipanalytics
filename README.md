**IP intelligence, VPN infrastructure research, ASN analytics, and GeoIP data engineering**

I build open datasets, parsers, dashboards, and automation around IP ranges, ASN metadata, MMDB/GeoIP databases, VPN infrastructure, Tor exits, crawler networks, and network reputation signals.

## Focus

- IP and ASN enrichment pipelines with source provenance and confidence scoring
- VPN, proxy, Tor, hosting, CDN, cloud, and crawler infrastructure analysis
- GeoIP/MMDB processing, validation, and multi-source location consensus
- Open network intelligence datasets for security, fraud detection, and routing analytics
- Dashboards and automation for repeatable IP intelligence workflows

---

## Featured Projects

| Project | Area | What it does |
|---|---|---|
| [GeoForge](https://github.com/ipanalytics/GeoForge) | GeoIP / MMDB | Builds a local IPv4 GeoIP database by merging DB-IP Lite, MaxMind GeoLite2, IP2Location LITE, Sypex Geo, GeoNames, geofeeds, RIR and WHOIS signals. |
| [IP-Knowledge-Layer](https://github.com/ipanalytics/IP-Knowledge-Layer) | IP enrichment | Open knowledge layer for CIDR, ASN, cloud, CDN, crawler, Tor, and VPN-adjacent network context with provenance and confidence. |
| [CrawlerScope](https://github.com/ipanalytics/CrawlerScope) | Dashboard | Interactive crawler IP intelligence dashboard for search, AI, and user-triggered fetchers. |
| [Tor-Radar](https://github.com/ipanalytics/Tor-Radar) | Tor intelligence | Tor network monitoring and relay/exit-node intelligence. |
| [vpn-provider-overlap-intelligence](https://github.com/ipanalytics/vpn-provider-overlap-intelligence) | VPN analytics | Aggregate VPN provider infrastructure overlap analysis: shared /24s, hosting dependency, and provider relationship clusters. |
| [ASN-VPN-Network-Intelligence](https://github.com/ipanalytics/ASN-VPN-Network-Intelligence) | ASN / VPN data | Publishes CSV datasets generated from VPN infrastructure collection and ASN/provider analysis. |
