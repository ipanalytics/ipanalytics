## Focus Areas

* IP and ASN enrichment pipelines with provenance and confidence scoring
* VPN, proxy, Tor, hosting, CDN, and crawler infrastructure analysis
* GeoIP/MMDB compilation, validation, and consensus-based enrichment
* Open operational datasets for fraud detection, SIEM enrichment, and network analytics
* Static dashboards and reproducible GitHub-native publishing workflows

## Design Principles

| Principle           | Description                                                          |
| ------------------- | -------------------------------------------------------------------- |
| Reproducibility     | Deterministic dataset generation and traceable inputs                |
| Operational Utility | Lightweight exports suitable for pipelines and local lookups         |
| Source Transparency | Preserve provenance and confidence metadata                          |
| Static Deployment   | Prefer GitHub-native publishing and browser-only dashboards          |
| Defensive Focus     | Infrastructure intelligence for analytical and operational workflows |
