Transparent network intelligence for security, fraud, OSINT, and telemetry workflows.

IP Analytics focuses on practical infrastructure intelligence: collecting public network metadata, normalizing inconsistent provider data, joining ASN and CIDR context, validating GeoIP/MMDB outputs, and publishing compact datasets that are easy to inspect, compare, and integrate.

IP and ASN enrichment

CIDR ranges, ASN ownership, hosting and CDN context, RIR and WHOIS metadata, geofeeds, source provenance, and confidence-aware enrichment.

Infrastructure visibility

VPN, proxy, Tor, hosting, cloud, CDN, crawler, AI fetcher, scanner, abuse, and reputation signals from auditable public and derived sources.

Operational datasets

CSV, JSON, JSONL, MMDB, static dashboards, static APIs, local lookup databases, reproducible builds, quality checks, and GitHub-native publishing workflows.

Featured active projects

VPN Infrastructure Intelligence Lab Aggregate VPN infrastructure intelligence dashboard for provider, country, ASN, relationship, archetype, and hosting dependency analysis without publishing raw endpoint data. ASN-Signal-Graph ASN-level infrastructure signal aggregation for VPN overlap, Tor visibility, public feed exposure, and defensive network analytics. IP-Knowledge-Layer Open knowledge layer for CIDR, ASN, cloud, CDN, crawler, Tor, and VPN-adjacent network context with source provenance and confidence. blackroute Security intelligence pipeline for hostile IP infrastructure, abuse feeds, anonymizers, attack telemetry, and runtime lookup databases. CrawlerScope Interactive crawler IP intelligence dashboard for search bots, AI crawlers, monitoring probes, scanners, and user-triggered fetchers. Tor-Radar Hourly public Tor relay data collection with compact snapshots and a browser-only dashboard. GeoForge Local IPv4 GeoIP database compiler combining DB-IP Lite, GeoLite2, IP2Location LITE, Sypex Geo, GeoNames, geofeeds, RIR, and WHOIS signals. MMDBForge Developer toolkit for inspecting, validating, diffing, and explaining custom MaxMind DB and GeoIP MMDB files. MMDB-WatchTower Production-safe MaxMind DB updater with verification, smoke tests, atomic swaps, rollback, systemd support, and Prometheus metrics.

Why this work matters

Auditable IP intelligence is easier to trust and operate.

Many IP intelligence feeds are useful but difficult to inspect. IP Analytics projects prioritize clear inputs, compact outputs, source-aware scoring, reproducible update paths, and operational formats that are easy to review, compare, and integrate.
